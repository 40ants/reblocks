==========================
 Contributing to Weblocks
==========================


1. Checkout branch.
2. Make changes.
3. Test changes::

     rove weblocks.asd
   
4. Add tests into the ``t/`` subdirectory.
5. Make a pull-request.
6. Discussion/review

See tutorial at http://trac.common-lisp.net/cl-weblocks/wiki/WeblocksDevelopment
